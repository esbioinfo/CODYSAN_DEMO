How does the diagnostic tool work?

CoDysAn is a webpage-based telemedicine tool that can help in the diagnostic of Congenital Dyserythropoyetic Anaemias (CDA). The diagnostic algorithm is transformed into a user-friendly form here the user, patient or doctor, needs to provide some information, including:

  • - Age
  • - Gender
  • - Hemoglobin level

Gender and age are used by the algorithm to know the normal values for the different medical information provided (hemoglobin level, reticulocytes, platelets, etc.). You can find the values used by the algorithm here. For simplification the algorithm, we only consider three different ages:

  • - From 0 to 6 months
  • - From 6 months to 12 years
  • - Older than 12 years old

If the hemoglobin level shows possible anemia, a new form appears asking the user to provide the following data:

  • - Mean Corpuscular Volume (MCV)
  • - Reticulocytes count
  • - Platelets count

Depending on the data provided, a new form will appear asking the user to exclude other possible cause(s) of anemias. The user needs to select at least one option to reach the next step. If requirements are met, the user is then asked to select other possible factors that may be causing the Congenital Dyseritropoyetic Anemia (CDA).

Finally, depending on the provided information the user will get a result containing the clinical suspicion (any of the CDA types) if it applies, or a brief explanation if there is no clinical suspicion of CDA.
